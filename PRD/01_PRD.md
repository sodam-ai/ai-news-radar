> 생성일: 2026-03-22

## 1. 제품 개요

### 한 줄 요약
AI 관련 뉴스·기술·도구 정보를 자동으로 수집하고 AI가 요약·분류해서 한눈에 볼 수 있는 개인용 정보 대시보드.

### 해결하는 문제
AI 시대에 쏟아지는 정보(새 모델 출시, 도구 업데이트, 논문, 트렌드)를 사람이 직접 수집하기엔 시간이 부족하다. 중요한 소식을 놓치거나, 수백 개 글을 일일이 읽어야 하는 비효율이 발생한다.

### 핵심 가치
- **자동 수집**: RSS·웹사이트·뉴스레터에서 AI가 알아서 긁어옴 — 내가 찾아다닐 필요 없음
- **AI 요약 + 자세히 보기**: 핵심 3줄 요약을 먼저 보여주고, 클릭하면 원문 전체를 펼쳐서 볼 수 있음
- **스마트 분류**: AI 도구/논문/트렌드/활용법 등 카테고리 자동 분류 — 관심 분야만 골라보기
- **중요도 점수**: AI가 각 뉴스에 ⭐1~5 중요도를 매겨서 "꼭 봐야 할 글" vs "나중에 봐도 되는 글" 구분
- **감성 분석**: 각 뉴스에 긍정/부정/중립 감성 태그 자동 부여 — "이 기술이 좋다는 글" vs "위험하다는 글" 한눈에 구분
- **중복 병합 + 다시각도 커버리지**: 같은 뉴스를 여러 매체가 보도하면 1개로 합치고, 클릭하면 각 매체의 시각을 비교해서 볼 수 있음
- **오늘의 브리핑**: 매일 "꼭 봐야 할 AI 뉴스 TOP 5"를 AI가 자동 작성 — 1분 안에 하루치 파악
- **키워드 워치리스트**: 내가 추적하는 기술(Claude, MCP 등)이 언급되면 자동 하이라이트
- **인앱 리더 뷰**: 외부 사이트로 나가지 않고 대시보드 안에서 깨끗하게 원문 읽기 + URL 클릭 시 새 탭으로 원문 바로 이동
- **배치 처리**: 글 5~10개를 API 1회로 처리 — 무료 한도 안에서 여유롭게 운영
- **15개+ 프리셋 소스**: 설치하자마자 검증된 AI 뉴스 소스(RSS + 뉴스레터)에서 수집 시작 — 설정 없이 바로 사용
- **노트 앱 연동**: 중요한 글을 Notion DB나 Obsidian 폴더로 바로 내보내기 — 뉴스가 지식으로 축적됨
- **AI 학습 필터링**: 내가 읽는 패턴을 AI가 학습해서 점점 더 정확한 추천을 해줌
- **AI 뉴스 채팅**: 수집된 뉴스에 대해 AI와 대화. "이번 주 Claude 관련 뉴스 요약해줘", "GPT-5와 Gemini 비교 뉴스 보여줘" 같은 자연어 질문
- **음성 브리핑**: 매일 아침 브리핑을 AI 음성으로 들을 수 있음 — 출근길에 눈 없이 AI 뉴스 파악 (NotebookLM 스타일)
- **뉴스 연결 지도**: 뉴스 간의 관계를 인터랙티브 그래프로 시각화 — "GPT-5 출시" → "OpenAI 투자" → "AI 규제 논의" 흐름 파악
- **뉴스 소스 자동 발견**: AI가 내 관심사 기반으로 새로운 AI 뉴스 소스를 자동 추천
- **스마트 모델 라우팅**: 단순 분류는 Flash-Lite(무료 1000회/일), 요약/브리핑은 Flash(250회/일)로 분리 — 무료 한도 3배 확장
- **Gemini Context Caching**: 시스템 프롬프트를 캐싱해서 토큰 비용 90% 절감 — 무료 한도 내에서 훨씬 더 많은 글 처리
- **멀티 플랫폼 허브**: 대시보드뿐 아니라 텔레그램/디스코드/슬랙 어디서든 뉴스 요약+브리핑+AI 채팅 가능 — 뉴스가 나를 찾아오는 구조
- **Gemini 멀티모달 분석**: 뉴스 속 이미지/차트/인포그래픽을 AI가 분석해서 요약에 반영 — "벤치마크 차트에서 수학 95점, 코딩 92점" 같은 시각 정보까지 추출
- **벡터 시맨틱 검색**: ChromaDB + Gemini Embedding 2로 "에이전트 관련 뉴스" 질문에 "자율 시스템", "도구 사용" 등 의미적으로 유사한 뉴스도 검색
- **크롬 확장 프로그램**: 웹 브라우징 중 AI 관련 글을 클릭 한 번으로 수집+분류+요약 — RSS에 없는 글도 수집
- **나만의 AI 뉴스레터 발행**: 수집된 뉴스로 주간 AI 뉴스레터를 자동 생성해서 팀/구독자에게 이메일 발행
- **뉴스 감성 온도계**: 감성 데이터를 시간별로 집계해서 "AI 업계 분위기" 시각화 — "오늘 AI 업계는 긍정적 72%"
- **GitHub Actions 서버리스**: 로컬 PC를 24시간 켜둘 필요 없이 GitHub Actions로 수집+처리 완전 자동화 (무료 2000분/월)
- **AI 타임캡슐**: "1개월 전 오늘 무슨 AI 뉴스가 있었지?" 회고 기능으로 장기 트렌드 파악
- **AI 멀티 에이전트 자율 운영**: 수집 에이전트→분석 에이전트→판단 에이전트→배포 에이전트가 자율 협업. MCP 표준으로 에이전트 간 통신. 사람은 전략만 결정
- **AI 팩트체크**: 뉴스의 정확성을 AI가 교차 검증. "이 주장은 3개 매체에서 확인됨 ✅" vs "1개 매체만 보도 ⚠️" 자동 표시
- **뉴스 디베이트 모드**: AI가 뉴스에 대해 찬성/반대 양쪽 시각을 자동 생성. "AI 규제 필요하다" 뉴스에 찬성 근거 3개 + 반대 근거 3개 정리
- **PWA 모바일 앱화**: Streamlit 대시보드를 PWA로 변환. 모바일 홈화면에 아이콘 추가, 푸시 알림, 오프라인 캐시
- **Ollama 로컬 LLM**: Gemini API 없이 로컬에서 AI 요약/분류 실행. 완전 오프라인, 무제한 요청, 데이터 외부 유출 제로
- **하이브리드 LLM 3단계 라우팅**: 단순 작업→Ollama(무제한), 중간→Flash-Lite(1000/일), 고품질→Flash(250/일). 비용 제로에 가까운 운영
- **대시보드 커스터마이징**: 위젯 배치를 자유롭게 구성. 브리핑을 왼쪽에, 타임라인을 오른쪽에, 감성 온도계를 상단에
- **자동 소셜 포스팅**: 중요 뉴스를 X/LinkedIn/블로그에 AI가 플랫폼별 최적화된 포스트를 자동 생성·게시
- **게이미피케이션**: AI 뉴스 퀴즈 + 데일리 스트릭 + 레벨/배지/리더보드. Duolingo 스타일로 뉴스 읽기 습관 형성. 7일 연속 읽기 불꽃 이모지 획득
- **주간 AI 인텔리전스 리포트**: 매주 자동 생성되는 임원용 "이번 주 AI 업계 핵심 트렌드 3개 + 예측 + 행동 제안" PDF. 팀 미팅에서 바로 공유
- **스마트 알림**: 단순 키워드가 아닌, AI가 "이 뉴스를 봐야 하는 이유: 당신이 추적 중인 Claude와 직접 관련" 같은 개인화된 이유와 함께 알림
- **AI 예측 엔진**: "다음 주에 핫이슈가 될 AI 주제 3개" 예측 + 나중에 맞았는지 자동 검증. 트렌드를 남보다 먼저 파악
- **AI 생태계 맵**: AI 기업/기술/인물 간의 관계를 인터랙티브 맵으로 시각화. "OpenAI" 클릭 → 관련 기업·인물·기술이 연결된 네트워크
- **경쟁사 모니터링**: 특정 AI 기업/제품을 추적 목록에 등록 → 해당 기업 관련 뉴스를 자동 수집+분석+"이번 주 OpenAI의 주요 움직임" 요약
- **AI 용어 사전**: 뉴스에서 나오는 AI 전문 용어를 자동 추출해서 쉽게 설명. "MCP란? → AI 앱들이 데이터에 접근하는 표준 방식(USB-C처럼)"
- **AI 팟캐스트 자동 생성**: NotebookLM 스타일로 AI 호스트 2명이 오늘의 AI 뉴스를 토론하는 팟캐스트를 매일 자동 생성. "오늘 GPT-5가 발표되었는데, 왜 중요하냐면..." "맞아, 흥미로운 건..."
- **플러그인 시스템**: 다른 개발자가 AI News Radar에 플러그인을 만들 수 있는 확장 생태계. "새로운 뉴스 소스 어댑터", "커스텀 분석 엔진" 등
- **팀 협업 모드**: 여러 사람이 같은 대시보드를 공유하고 뉴스에 코멘트/태그/투표. 팀 큐레이션으로 더 정확한 뉴스 선별
- **AI 콘텐츠 자동 생성**: 수집된 뉴스 기반으로 블로그 포스트/트윗 스레드/LinkedIn 글을 AI가 내 톤앤매너로 자동 작성
- **기존 서비스와 차이**: Daily.dev, Feedly 같은 서비스는 범용이지만, 이건 AI 분야에 특화 + **AI 에이전트가 자율적으로 수집·분석·검증·예측·배포하고, 팟캐스트로 들려주고, 콘텐츠를 생산하고, 플러그인으로 확장되는** 세계 유일의 AI 뉴스 인텔리전스 플랫폼

## 2. 사용자

### 주요 사용자
- **누구**: AI에 관심 있는 개발자/기획자/연구자 (처음엔 나 혼자)
- **상황**: 아침에 출근 전, 점심시간, 퇴근 후 — AI 업계 동향을 빠르게 훑고 싶을 때
- **목표**: 중요한 AI 소식을 놓치지 않고, 최소 시간으로 최대 정보를 얻고 싶다

### 사용자 시나리오
1. 아침에 AI News Radar 대시보드를 열면
2. 밤사이 자동 수집된 AI 뉴스가 카테고리별로 정리되어 있고
3. AI가 요약한 핵심 3줄을 훑어보고, 관심 가는 글만 원문 링크로 이동한다

## 4. 사용자 흐름 (User Flow)

### 핵심 흐름
```
소스 등록 -> 자동 수집 -> AI 요약·분류·감성 -> 대시보드(실시간) -> 읽기/채팅/음성 -> 지식 축적
```

### 상세 흐름
1. **첫 실행**: 기본 프리셋 15개+ 소스가 자동 등록됨 (RSS + 뉴스레터, 추가/삭제 가능)
2. **자동 수집**: 스케줄러가 설정된 주기로 새 글을 크롤링 + 뉴스레터 이메일 자동 파싱
3. **AI 처리**: Gemini API 배치로 → 요약 3줄 + 카테고리 분류 + ⭐중요도 + 😊감성 분석 + 중복 감지
4. **중복 병합**: 같은 뉴스가 여러 소스에서 나오면 대표 1개로 합치고 "▶ 3개 매체 비교 보기" 표시
5. **대시보드 확인**: ⭐중요도 높은 순으로 정렬 + 카테고리/감성 필터로 관심 분야만 보기
6. **요약 → 자세히**: 3줄 요약을 훑어보고, 관심 가는 글은 펼쳐서 전문 보기
7. **원문 이동**: 제목이나 URL 클릭 시 새 탭으로 원문 사이트 바로 이동
8. **지식 축적**: 중요 글을 Notion/Obsidian으로 내보내서 나만의 AI 지식 베이스에 저장

## 5. 성공 기준

- [ ] 최소 10개 이상의 AI 뉴스 소스에서 자동 수집이 동작한다
- [ ] AI 요약이 원문의 핵심을 정확하게 반영한다 (주관 평가)
- [ ] 카테고리 자동 분류 정확도가 체감상 80% 이상이다
- [ ] 하루 AI 뉴스를 5분 안에 훑어볼 수 있다
- [ ] 앱이 안정적으로 24시간 돌아간다 (크래시 없이)
