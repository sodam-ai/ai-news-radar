## AI News Radar란?

AI 분야는 매시간 변합니다. 새로운 모델 출시, 도구 업데이트, 논문 발표, 기업 동향 — 모두 수십 개의 분산된 소스에 흩어져 있습니다. AI News Radar는 이 소음을 제거합니다. 74개의 엄선된 소스에서 뉴스를 지속적으로 수집하고, AI가 모든 기사를 요약, 분류, 점수화, 팩트체크한 뒤 — 깔끔하고 실용적인 브리핑으로 전달합니다.

**한 문장으로:** 74개 사이트를 직접 방문하지 마세요. AI가 전부 읽고 중요한 것만 알려줍니다.

## 🔒 보안 & 개인정보 — 반드시 읽으세요

### API 키는 100% 개인 소유 — GitHub에 절대 업로드되지 않습니다

이 프로젝트를 다운로드하면 **API 키, 비밀번호, 개인 정보가 전혀 포함되어 있지 않습니다.**

작동 방식:
- 본인 컴퓨터에 `.env`라는 파일을 직접 만듭니다
- 본인이 발급받은 API 키를 그 파일에 넣습니다 (Google/Groq 등에서 무료로 발급)
- 그 파일은 **절대 인터넷에 업로드되지 않습니다** — `.gitignore`가 차단합니다
- 이 앱을 다운로드한 모든 사람은 **자신의 키**로 **자신의 할당량**을 사용합니다

### 안전하다는 증거

| 확인 항목 | 결과 |
|-----------|------|
| GitHub 저장소에 `.env` 파일 존재 | ❌ 없음 (.gitignore로 차단됨) |
| 소스 코드에 API 키 하드코딩 | ❌ 없음 — 모든 코드가 환경변수에서 읽음 |
| 내 키가 다른 사람과 공유됨 | ❌ 불가능 — 각자 자신의 키 설정 |
| AI 제공업체 외 서버 연결 | ❌ 뉴스 RSS 피드와 선택한 AI API에만 연결 |

### 다운로드한 사람이 해야 할 일

이 앱을 다운로드한 모든 사람은:
1. 본인의 무료 API 키를 발급받습니다 (10초, 신용카드 불필요)
2. 본인의 `.env` 파일을 만들고 키를 넣습니다
3. 끝 — 자신의 할당량을 사용, 나의 할당량과 완전히 분리됨

## 시작하기 — 완전 초보자 가이드 (단계별)

> **코딩 경험 전혀 없어도 됩니다.** 복사 붙여넣기만 할 수 있으면 설정할 수 있습니다. 각 단계를 천천히 따라하세요.

### Step 1 — Python 설치

Python은 이 앱이 만들어진 프로그래밍 언어입니다. 한 번만 설치하면 됩니다.

1. 주소 입력: **https://www.python.org/downloads/**
2. 큰 노란색 **"Download Python"** 버튼 클릭
3. 다운로드된 파일 실행 (Windows에서 `.exe` 파일)
4. ⚠️ **매우 중요:** 설치 화면 하단의 **"Add Python to PATH"** 체크박스를 반드시 체크
5. **"Install Now"** 클릭
6. 완료될 때까지 기다린 후 "Close" 클릭

**확인 방법:** `Win + R` 누르기 > `cmd` 입력 > Enter. 검은 창에서 다음을 입력:
```
python --version
```
`Python 3.11.9` 같은 내용이 보이면 설치 성공입니다.

---

### Step 2 — 프로젝트 다운로드

**방법 A: 직접 다운로드 (가장 쉬움)**
1. 주소 입력: **https://github.com/sodam-ai/ai-news-radar**
2. 초록색 **"Code"** 버튼 클릭
3. **"Download ZIP"** 클릭
4. 다운로드된 ZIP 파일 찾기 (보통 다운로드 폴더)
5. 우클릭 > **"모두 압축 풀기"** > 폴더 선택 > "압축 풀기"
6. `ai-news-radar-main` 폴더가 생성됩니다

---

### Step 3 — 터미널 열기

1. 압축 해제된 폴더 열기 (`app.py` 파일이 있는 폴더)
2. 파일 탐색기 상단 주소창 클릭
3. `cmd` 입력 후 Enter
4. 검은 터미널 창이 올바른 폴더에서 열립니다 ✅

---

### Step 4 — 필수 패키지 설치

터미널 창에 다음을 입력하고 Enter:
```bash
pip install -r requirements.txt
```
완료될 때까지 기다립니다 (1~3분). 많은 텍스트가 스크롤되는 것은 정상입니다.

---

### Step 5 — 무료 API 키 발급

AI 기능을 사용하려면 무료 API 키가 필요합니다. 가장 빠른 방법:

**방법 A: Google Gemini (추천 — 하루 1,000회 무료)**
1. 주소 입력: **https://aistudio.google.com/apikey**
2. Google 계정으로 로그인
3. **"Create API Key"** 클릭
4. **"Create API key in new project"** 클릭
5. 나타나는 키 복사 (`AIzaSy...` 로 시작)

**방법 B: Groq (하루 14,400회 무료, 더 빠름)**
1. 주소 입력: **https://console.groq.com/keys**
2. Google로 가입 (10초)
3. **"Create API Key"** 클릭
4. 키 복사 (`gsk_` 로 시작)

> ✅ **신용카드 불필요.** 어떤 방법도 요금이 부과되지 않습니다.

---

### Step 6 — API 키 설정

1. 프로젝트 폴더에서 **`.env.example`** 파일 찾기
2. 그 파일을 복사하여 이름을 **`.env`** 로 변경 (`.example` 부분 삭제)
   - Windows: 우클릭 > 복사 > 붙여넣기 > 이름 변경
3. `.env` 파일을 메모장으로 열기 (우클릭 > 연결 프로그램 > 메모장)
4. 선택한 플랫폼의 줄을 찾아 키 입력:

**Gemini 키를 받은 경우:**
```
GEMINI_API_KEY=AIzaSy_여기에_키_붙여넣기
```

**Groq 키를 받은 경우:**
```
GROQ_API_KEY=gsk_여기에_키_붙여넣기
```

5. 저장 후 닫기

> 🔒 **키는 안전합니다.** 이 `.env` 파일은 본인 컴퓨터에만 존재합니다. GitHub나 다른 사람에게 절대 업로드되지 않습니다.

---

### Step 7 — 앱 실행

**웹 모드 (브라우저에서 열림):**
```bash
streamlit run app.py
```
브라우저가 자동으로 **http://localhost:6601** 을 엽니다

**데스크톱 모드 (독립 창):**
```bash
python desktop.py
```

**Windows 바로가기:** 프로젝트 폴더의 **`AI_News_Radar.bat`** 더블클릭

### 첫 사용 — 실행 후 할 일

1. 사이드바(왼쪽 패널)에서 **"📡 Collect"** 클릭 — 74개 소스에서 뉴스 수집 (~1분 소요)

## 자주 묻는 문제 해결

| 문제 | 해결 방법 |
|------|----------|
| `python` 명령어를 찾을 수 없음 | **"Add to PATH"** 체크 후 Python 재설치 |
| `pip` 명령어를 찾을 수 없음 | `python -m pip install -r requirements.txt` 시도 |
| `streamlit` 명령어를 찾을 수 없음 | `pip install streamlit` 실행 |
| "API 키 미설정" 경고 | Step 6에 따라 API 키가 있는 `.env` 파일 생성 |
| 기사가 나타나지 않음 | "Collect" 먼저 클릭, 1분 기다린 후 "AI Process" |
| AI 기능이 작동하지 않음 | `.env` 파일이 존재하고 유효한 API 키가 있는지 확인 |
| 카테고리가 0개 | `python scripts/reclassify.py` 실행 |
| 포트 6601 이미 사용 중 | `streamlit run app.py --server.port 7777` |
| 수집이 느림 | 정상 — 74개 소스는 약 60초 소요 |

**API 키 관련 자주 하는 실수:**
- 키 앞뒤에 공백이 있는 경우 → 공백 모두 제거
- 만료된 키 사용 → 새로운 키 발급
- `.env` 파일 저장을 잊은 경우 → 메모장에서 저장 클릭
