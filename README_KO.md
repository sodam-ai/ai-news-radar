# AI News Radar

> **AI 뉴스를 자동으로 수집·요약·분류해주는 나만의 AI 뉴스 대시보드 — 74개 소스, 35개 LLM 지원**

**[English](./README.md) / Korean / [Japanese](./README_JA.md) / [Chinese](./README_ZH.md)**

---

## 이게 뭔가요?

AI News Radar는 전 세계 AI 뉴스를 **자동으로 모아서**, AI가 **요약하고, 분류하고, 중요도를 매겨주는** 개인용 대시보드입니다. **35개 LLM 플랫폼** 지원(대부분 무료), **웹+데스크톱** 동시 사용, **5개 SNS**에 자동 게시까지 가능합니다.

**쉽게 말하면:** 매일 수십 개 AI 뉴스 사이트를 돌아다니는 대신, 이 앱이 대신 해주고 중요한 것만 보여줍니다.

## 주요 기능 (45개 이상)

### 대시보드
| 기능 | 설명 |
|------|------|
| 오늘의 브리핑 | AI가 매일 "오늘의 AI 뉴스 TOP 5" 자동 생성 |
| 관심 분야 브리핑 | 이미지/영상, 바이브코딩, 온톨로지 분야별 맞춤 브리핑 |
| 카테고리 필터 | 9개 카테고리별로 클릭 한 번에 필터 (건수 표시) |
| 감성 온도계 | 긍정/중립/부정 비율 Plotly 차트 |
| 음성 브리핑 | 브리핑을 한국어 AI 음성으로 듣기 (여성/남성 선택) |
| 주간 리포트 | 핵심 트렌드 + 분야별 동향 + 전망 자동 생성 |
| 뉴스레터 | 일간/주간 브리핑을 이메일로 자동 발송 |

### 뉴스피드
| 기능 | 설명 |
|------|------|
| 74개 소스 수집 | 전 세계 74개 RSS 소스에서 자동 수집 |
| AI 요약 | 각 기사를 한국어 3줄로 요약 |
| 9개 카테고리 | 도구, 연구, 트렌드, 튜토리얼, 비즈니스, **이미지/영상**, **바이브코딩**, **온톨로지**, 기타 |
| 팩트체크 배지 | "✅ 3개 매체 확인" vs "⚠️ 단독 보도" |
| 중복 병합 | 같은 뉴스 자동 합침 |
| 검색 | 키워드 + 카테고리 + 감성 + 읽음 상태 필터 |
| 북마크 + 메모 | 중요 기사 저장 + 메모 추가 |
| 페이지네이션 | 10개씩 페이지 넘김 |

### AI 기능
| 기능 | 설명 |
|------|------|
| AI 채팅 | 수집된 뉴스에 대해 자연어로 질문 |

| 도구 비교 | 21개 AI 도구 뉴스 언급량/감성 비교 차트 |
| 트렌드 차트 | 키워드별 시계열 언급량 + 급상승 키워드 |

## 시작하기 (완전 초보자용)

> **코딩 경험이 전혀 없어도 됩니다.** 차근차근 따라하세요.

### 1단계: Python 설치

1. [python.org/downloads](https://www.python.org/downloads/) 접속
2. 큰 노란색 **"Download Python"** 버튼 클릭
3. 다운로드된 파일 실행
4. **중요:** 하단의 **"Add Python to PATH"** 반드시 체크!
5. **"Install Now"** 클릭

**확인:** 명령 프롬프트(`Win + R` > `cmd` > Enter)에서:
```
python --version
```

### 2단계: 프로젝트 다운로드

**방법 A: Git (추천)**
```
git clone https://github.com/sodam-ai/ai-news-radar.git
cd ai-news-radar
```

**방법 B: 직접 다운로드**
1. [GitHub 페이지](https://github.com/sodam-ai/ai-news-radar) 접속
2. 초록색 **"Code"** 버튼 > **"Download ZIP"** 클릭
3. ZIP 파일 압축 해제

### 3단계: 패키지 설치

프로젝트 폴더에서 명령 프롬프트 열고:
```
pip install -r requirements.txt
```
14개 패키지가 자동 설치됩니다. 완료될 때까지 기다리세요.

### 4단계: API 키 발급 (무료)

**Groq 추천 (가장 쉬움):**
1. [console.groq.com/keys](https://console.groq.com/keys) 접속
2. Google 계정으로 가입
3. **"Create API Key"** 클릭
4. 키 복사 (`gsk_`로 시작)

### 5단계: API 키 설정

1. 프로젝트 폴더에서 `.env.example` 파일 찾기
2. 복사해서 이름을 `.env`로 변경
3. 메모장으로 `.env` 열기
4. 키 입력:
```
GROQ_API_KEY=gsk_여기에실제키입력
```
5. 저장 후 닫기

> **보안:** `.env` 파일은 GitHub에 자동으로 제외됩니다. 절대 공유하지 마세요.

### 6단계: 앱 실행

**웹 모드:**
```
streamlit run app.py
```
브라우저가 **http://localhost:6601** 을 자동으로 엽니다.

**데스크톱 모드:**
```
python desktop.py
```
또는 `AI_News_Radar.bat` 더블클릭

### 7단계: 첫 사용

1. 사이드바 **"🔄 수집"** 클릭 → 74개 소스에서 뉴스 수집
2. **"🤖 AI 처리"** 클릭 → 모든 기사 AI 분석
3. **"📋 브리핑 생성"** 클릭 → 오늘의 TOP 5 생성
4. 대시보드에서 카테고리별로 탐색!

## 사용 가이드

| 하고 싶은 것 | 방법 |
|-------------|------|
| 오늘 요약 보기 | 대시보드 탭 > 브리핑 |
| 이미지/영상 뉴스만 보기 | 대시보드 탭 > "AI 이미지/영상" 클릭 |
| 특정 주제 검색 | 뉴스피드 탭 > 검색 뷰 > 키워드 입력 |
| AI에게 질문 | AI 탭 > 채팅 뷰 > 질문 입력 |
| AI 도구 비교 | 인사이트 탭 > 도구 비교 |
| 트렌드 보기 | 인사이트 탭 > 트렌드 |
| SNS 콘텐츠 생성 | 공유 탭 > 콘텐츠 생성 > 기사+플랫폼 선택 |
| SNS에 게시 | 공유 탭 > SNS 게시 > 플랫폼 선택 > 게시 |
| 음성으로 듣기 | 대시보드 탭 > 음성 선택 > "음성" 클릭 |

## 문제 해결

| 문제 | 해결 |
|------|------|
| `pip` 못 찾음 | Python 재설치, "Add to PATH" 체크 |
| "API 키 미설정" 경고 | `.env` 파일에 API 키 입력 |
| 기사 안 나옴 | "수집" → "AI 처리" 순서로 클릭 |
| 카테고리에 기사 없음 | `python scripts/reclassify.py` 실행 |
| 포트 사용 중 | `streamlit run app.py --server.port 7429` |
| PDF 실패 | Windows 전용 (한글 폰트) |
