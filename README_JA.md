## AI News Radarとは？

AI News Radarは、世界中のAI関連ニュースを**自動的に収集**し、AIが**要約・分類・重要度を付ける**個人用ダッシュボードです。OpenAI、Gemini、Groq、Claudeなど**35のLLMプラットフォーム**に対応し、ほとんどが無料で利用できます。

**簡単に言うと：** 毎日15以上のニュースサイトを巡回する代わりに、このアプリが代わりにやって、重要なものだけを表示します。

### Phase 1 — コア（17機能）

| 機能 | 説明 |
|------|------|
| 自動収集 | 15のプリセットソースからニュースを自動収集 |

| 重要度スコア | 各記事に1〜5つ星の評価 |
| 感情分析 | 各記事をポジティブ/ニュートラル/ネガティブに分類 |
| 今日のブリーフィング | 毎日「今日のAIニュースTOP 5」を自動生成 |
| 重複統合 | 同じニュースを複数メディアが報道した場合、1つにまとめる |
| キーワードウォッチリスト | 追跡キーワードを含むニュースをハイライト |
| アプリ内リーダー | 広告なしでダッシュボード内で原文を閲覧 |
| ダークモード | ダーク/ライトテーマの切り替え |
| エクスポート（Markdown/PDF） | ブリーフィングと記事をダウンロード |
| 画像分析 | ニュース内のチャート/インフォグラフィックをAIが分析 |
| リアルタイム更新 | 5分ごとに自動更新 |
| タイムラインビュー | 時系列でニュースを閲覧 |
| Context Caching | APIトークンコスト削減 |
| スマートルーティング | タスクに応じてモデルを自動選択 |

## はじめ方（初心者向けステップバイステップ）

> **プログラミング経験は不要です。**

### ステップ1：Pythonインストール
[python.org/downloads](https://www.python.org/downloads/) → **"Add Python to PATH"** を必ずチェック

### ステップ2：プロジェクトダウンロード
```
git clone https://github.com/sodam-ai/ai-news-radar.git
cd ai-news-radar
```

### ステップ3：パッケージインストール
```
pip install -r requirements.txt
```

### ステップ4：APIキー取得（無料）
上記テーブルから1つ選択 → キーをコピー

### ステップ5：APIキー設定
`.env.example` を `.env` にコピー → メモ帳で開く → キーを入力

### ステップ6：アプリ起動
```
streamlit run app.py
```
ブラウザが **http://localhost:6601** を自動的に開きます。

## トラブルシューティング

| 問題 | 解決方法 |
|------|----------|
| `pip` が見つからない | Python再インストール、"Add to PATH"チェック |
| 「LLM APIキー未設定」警告 | `.env`にAPIキーを最低1つ入力 |
| 記事が表示されない | 「収集」→「AI処理」の順にクリック |
| ポート使用中 | `--server.port 6602` オプション使用 |
| PDF失敗 | Windows専用（韓国語フォント使用） |
