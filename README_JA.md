## AI News Radarとは？

AIの世界は毎時間動いています。新モデルのリリース、ツールの更新、論文の発表、企業の方針転換 — それらはすべて、数十の散らばったサイトに分散しています。**AI News Radar**はそのノイズを一掃します。**74の厳選されたソース**からニュースを継続的に収集し、AIがすべての記事を要約・分類・重要度評価・ファクトチェックしたうえで、すっきりとした実用的なブリーフィングとして届けます。

**一言で言えば:** 74のサイトを自分で巡回するのはやめましょう。AIがすべて読んで、大事なことだけ教えてくれます。

## 🔒 セキュリティ＆プライバシー — 最初にお読みください

**このプロジェクトを使い始める前に、必ずこのセクションをお読みください。**

### このプロジェクトにAPIキーは含まれていません

- **GitHubで公開されているコードには、APIキーが一切含まれていません。**
- `.env` ファイルは `.gitignore` に登録されているため、**GitHubには絶対にアップロードされません。**
- あなたのAPIキーは**あなたのパソコンの中だけ**に保存されます。インターネット上に流出することはありません。

### 各自でAPIキーを取得する必要があります

- **このプロジェクトを動かすには、あなた自身がAPIキーを取得する必要があります。**
- 他の人のAPIキーを使うことはできません。各ユーザーが自分専用のキーを使います。
- 無料で取得できるサービスがあります（後のステップで詳しく説明します）。

### APIキーの安全な管理ルール

| 正しい使い方 | やってはいけないこと |
|-------------|---------------------|
| `.env` ファイルに保存する | APIキーをソースコードに直接書き込む |
| `.env` ファイルを自分のPCだけに保管する | `.env` ファイルをSNSやチャットで他人に送る |
| 使わなくなったキーは無効化する | GitHubなどの公開リポジトリにアップロードする |
| 信頼できるサービスのキーだけを使う | スクリーンショットにAPIキーを写し込む |

> **安心してください:** `.gitignore` の設定により、`.env` ファイルがGitHubに誤ってアップロードされることはありません。あなたのAPIキーは、あなたのパソコンの中にだけ存在します。

### ステップ 1 — Pythonをインストールする

Pythonとは、このアプリを動かすためのプログラミング言語です。まずこれをパソコンにインストールします。

**手順:**

1. ブラウザ（ChromeやEdgeなど）を開き、アドレスバーに `python.org/downloads` と入力してアクセスする
2. ページの中央にある大きな黄色い **「Download Python」** ボタンをクリックする
3. ダウンロードが完了したら、ダウンロードされた `.exe` ファイルをダブルクリックして実行する
4. インストール画面が開きます。**【ここが最重要ポイント】** 画面の一番下に **「Add Python to PATH」** というチェックボックスがあります。**必ずこれにチェックを入れてください。** チェックを入れ忘れると、後の手順でエラーになります。
5. **「Install Now」** をクリックしてインストールが完了するまで待つ（1〜2分）

**インストールされたか確認する:**

`Win + R` キーを同時に押すと、小さなウィンドウが開きます。そこに `cmd` と入力してEnterを押します。黒い画面（コマンドプロンプト）が開いたら、次のコマンドを入力してEnterを押してください:

```bash
python --version
```

`Python 3.11.9` のような文字が表示されれば成功です。エラーが表示された場合は、Pythonのインストールをやり直し、「Add Python to PATH」のチェックを忘れずに入れてください。

5. 展開先のフォルダを選んで「展開」をクリックする

### ステップ 3 — 仮想環境を作成する（推奨）

仮想環境とは、このアプリ専用の作業スペースを作るものです。他のPythonアプリと設定が混ざらないよう保護してくれます。難しそうに聞こえますが、コマンド1つで完了します。

コマンドプロンプトまたはターミナルで次のコマンドを実行します:

```bash
python -m venv venv
```

次に、作成した仮想環境を「有効化」します。お使いのOSに合わせて実行してください:

```bash
# Windowsの場合
venv\Scripts\activate

# macOSまたはLinuxの場合
source venv/bin/activate
```

コマンドプロンプトの行頭に `(venv)` という文字が現れれば成功です。これ以降のすべての操作は、この状態で行ってください。

> **注意（Windowsのみ）:** 「実行ポリシー」エラーが出た場合は、PowerShellで `Set-ExecutionPolicy RemoteSigned -Scope CurrentUser` を実行してからもう一度試してください。

### ステップ 6 — APIキーをアプリに設定する

取得したAPIキーをアプリに教えます。

1. プロジェクトフォルダ（`ai-news-radar`フォルダ）の中を見ると、**`.env.example`** というファイルがあります
   - **表示されない場合:** Windowsの「エクスプローラー」で「表示」→「隠しファイル」にチェックを入れてください
2. **`.env.example`** ファイルを右クリックして「コピー」を選択し、同じフォルダに「貼り付け」する
3. 貼り付けたファイルの名前を **`.env`** に変更する
   - Windowsでは `.` から始まるファイル名に変更すると確認ダイアログが出る場合がありますが、「はい」を選んでください
4. **`.env`** ファイルをメモ帳（またはお好きなテキストエディタ）で開く
5. 取得したAPIキーを次のように貼り付けます:

```env
# Geminiを使う場合（コピーしたキーを = の後に貼り付け）:
GEMINI_API_KEY=AIzaSy...ここにGeminiキーを貼り付ける

# Groqを使う場合:
GROQ_API_KEY=gsk_...ここにGroqキーを貼り付ける

# OpenAIを使う場合（有料）:
# OPENAI_API_KEY=sk-...ここにOpenAIキーを貼り付ける
```

6. ファイルを上書き保存して閉じる

> **セキュリティの保証:** `.env` ファイルは `.gitignore` に登録されており、GitHubには絶対にアップロードされません。このファイルはあなたのパソコンの中だけに存在し、他人には見えません。他の人に見せたり、SNSに投稿したりしないようにしましょう。

### ステップ 7 — アプリを起動する

いよいよアプリを起動します！

**Webモード（ブラウザで開く — 推奨）:**

```bash
streamlit run app.py
```

コマンドを実行すると、自動的にブラウザが開き **http://localhost:6601** が表示されます。ブラウザが自動で開かない場合は、手動でアドレスバーに `http://localhost:6601` と入力してください。

**デスクトップモード（ネイティブウィンドウ）:**

```bash
python desktop.py
```

**もっと簡単な方法（Windowsのみ）:**

プロジェクトフォルダの中にある **`AI_News_Radar.bat`** ファイルをダブルクリックするだけで起動できます。

## SNSプラットフォームの設定

| プラットフォーム | 設定時間 | 難易度 | 設定方法 |
|-----------------|:--------:|:------:|---------|
| Discord | 30秒 | とても簡単 | チャンネル設定でWebhook URLを作成する |
| Telegram | 2分 | 簡単 | @BotFatherでボットを作成する |

| Threads | 10分 | 普通 | Meta開発者ポータルで設定する |

## よくある問題と解決法

| 問題 | 解決方法 |
|------|---------|
| `python` コマンドが見つからない | **「Add to PATH」** にチェックを入れてPythonを再インストールする |
| `pip` コマンドが見つからない | 代わりに `python -m pip install -r requirements.txt` を使う |
| `streamlit` が見つからない | `pip install streamlit` を実行し、仮想環境が有効化されているか確認する |
| 「APIキーが設定されていません」という警告 | `.env` ファイルを作成してAPIキーを1つ以上入力する（ステップ6参照） |
| 記事が表示されない | **「収集」** を先にクリックしてから **「AI処理」** をクリックする |
| カテゴリーに記事が0件 | `python scripts/reclassify.py` を実行して既存の記事を再分類する |
| ポート6601が使用中 | `streamlit run app.py --server.port 7777` を使う |
| macOS/LinuxでPDFエクスポートが失敗する | `NanumGothic` フォントをインストールする |
| デスクトップモードが起動しない | `pip install pywebview` が正しくインストールされているか確認する |
| 収集が遅い | 正常です — 74ソースを15並列ワーカーで収集するのに約60秒かかります |
| edge-ttsの音声エラー | インターネット接続を確認する（edge-ttsはオンライン接続が必要） |
| `.env` ファイルが作成できない | メモ帳で新規作成して `.env` という名前で保存する（拡張子 `.txt` にならないよう注意） |
| 仮想環境の有効化でエラーが出る | PowerShellで `Set-ExecutionPolicy RemoteSigned -Scope CurrentUser` を実行してから再試行する |
