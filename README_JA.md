## AI News Radarとは

AI分野は刻一刻と変化しています。新しいモデルのリリース、ツールのアップデート、論文の発表、企業の動向 — すべてが数十のサイトに散在しています。AI News Radarはこのノイズを解消します。74の厳選されたソースからニュースを継続的に収集し、AIがすべての記事を要約・分類・スコアリング・ファクトチェックした上で、分かりやすく実用的なブリーフィングとして届けます。

**一言で:** 74のサイトを毎日巡回する必要はありません。AIがすべてを読み、重要な情報だけをお知らせします。

## セキュリティとプライバシー — 最初にお読みください

### APIキーは100%個人所有 — アップロードされることはありません

このプロジェクトをダウンロードしても、**APIキー、パスワード、個人情報は一切含まれていません。**

仕組み:
- 自分のパソコンに`.env`というファイルを作成します
- 自分で取得したAPIキーをそのファイルに記入します（Google/Groqなどから無料で取得可能）
- そのファイルは**絶対にインターネットにアップロードされません** — `.gitignore`がブロックします
- このアプリをダウンロードした全員が、**自分のキー**で**自分の使用枠**を使います

### 安全性の証明

| 確認項目 | 結果 |
|----------|------|
| GitHubリポジトリに`.env`ファイル | 存在しない（.gitignoreでブロック済み） |
| ソースコードにAPIキーのハードコード | なし — すべて環境変数から読み取り |
| 自分のキーが他人と共有される | 不可能 — 各自が自分のキーを設定 |
| AIプロバイダ以外のサーバーへ接続 | ニュースRSSフィードと選択したAI APIにのみ接続 |

### ダウンロードした方がやるべきこと

このアプリをダウンロードしたすべての方は:
1. 自分の無料APIキーを取得します（10秒、クレジットカード不要）
2. 自分の`.env`ファイルを作成し、キーを記入します
3. 以上 — 自分の使用枠を使い、他のユーザーとは完全に分離されています

## はじめに — 完全初心者ガイド（ステップバイステップ）

> **プログラミング経験はゼロでOKです。** コピー&ペーストができれば設定できます。各ステップをゆっくり順番に進めてください。

### Step 1 — Pythonのインストール

Pythonはこのアプリを動かすプログラミング言語です。一度だけインストールすれば大丈夫です。

1. 以下のURLにアクセス: **https://www.python.org/downloads/**
2. 大きな黄色い **「Download Python」** ボタンをクリック
3. ダウンロードされたファイルを実行（Windowsでは`.exe`ファイル）
4. **非常に重要:** インストール画面の下部にある **「Add Python to PATH」** のチェックボックスを必ずオン
5. **「Install Now」** をクリック
6. 完了するまで待ち、「Close」をクリック

**確認方法:** `Win + R`を押す > `cmd`と入力 > Enter。黒い画面で以下を入力:
```
python --version
```
`Python 3.11.9`のように表示されればインストール成功です。

---

### Step 2 — プロジェクトのダウンロード

**方法A: 直接ダウンロード（最も簡単）**
1. 以下のURLにアクセス: **https://github.com/sodam-ai/ai-news-radar**
2. 緑色の **「Code」** ボタンをクリック
3. **「Download ZIP」** をクリック
4. ダウンロードされたZIPファイルを探す（通常はダウンロードフォルダ）
5. 右クリック > **「すべて展開」** > フォルダを選択 > 「展開」
6. `ai-news-radar-main`フォルダが作成されます

**方法B: Git Clone（Gitがインストール済みの場合）**
```bash
git clone https://github.com/sodam-ai/ai-news-radar.git
cd ai-news-radar
```

---

### Step 3 — ターミナルを開く

1. 展開されたフォルダを開く（`app.py`ファイルがあるフォルダ）
2. エクスプローラー上部のアドレスバーをクリック
3. `cmd`と入力してEnterを押す
4. 黒いターミナル画面が正しいフォルダで開きます

---

### Step 4 — 必要なパッケージのインストール

ターミナル画面で以下を入力してEnter:
```bash
pip install -r requirements.txt
```
完了するまで待ちます（1〜3分）。大量のテキストがスクロールされますが正常です。

---

### Step 5 — 無料APIキーの取得

AI機能を使うには無料のAPIキーが必要です。最も手軽な方法:

**方法A: Google Gemini（推奨 — 1日1,000回無料）**
1. 以下のURLにアクセス: **https://aistudio.google.com/apikey**
2. Googleアカウントでログイン
3. **「Create API Key」** をクリック
4. **「Create API key in new project」** をクリック
5. 表示されるキーをコピー（`AIzaSy...`で始まります）

**方法B: Groq（1日14,400回無料、高速）**
1. 以下のURLにアクセス: **https://console.groq.com/keys**
2. Googleで登録（10秒）
3. **「Create API Key」** をクリック
4. キーをコピー（`gsk_`で始まります）

> **クレジットカードは不要です。** どちらの方法でも料金は一切発生しません。

---

### Step 6 — APIキーの設定

1. プロジェクトフォルダで **`.env.example`** ファイルを探す
2. そのファイルをコピーして、名前を **`.env`** に変更（`.example`の部分を削除）
   - Windows: 右クリック > コピー > 貼り付け > 名前の変更
3. `.env`ファイルをメモ帳で開く（右クリック > プログラムから開く > メモ帳）
4. 選択したプロバイダの行を見つけてキーを入力:

**Geminiキーを取得した場合:**
```
GEMINI_API_KEY=AIzaSy_ここにキーを貼り付け
```

**Groqキーを取得した場合:**
```
GROQ_API_KEY=gsk_ここにキーを貼り付け
```

5. 保存して閉じる

> **キーは安全です。** この`.env`ファイルは自分のパソコンにのみ存在します。GitHubや他の人に共有されることは絶対にありません。

---

### Step 7 — アプリの起動

**Webモード（ブラウザで開きます）:**
```bash
streamlit run app.py
```
ブラウザが自動的に **http://localhost:6601** を開きます

**デスクトップモード（独立ウィンドウ）:**
```bash
python desktop.py
```

**Windowsショートカット:** プロジェクトフォルダの **`AI_News_Radar.bat`** をダブルクリック

---

### 初回起動後にやること

1. サイドバー（左パネル）の **「Collect」** をクリック — 74ソースからニュースを収集します（約1分）
2. **「AI Process」** をクリック — AIがすべての記事を分析します（APIキーが必要）
3. **「Briefing」** をクリック — 今日のTOP 5要約を生成します
4. 5つのタブを探索: ダッシュボード / ニュースフィード / AI / インサイト / 共有

> **ヒント:** APIキーがなくても「Collect」をクリックすればニュース一覧を閲覧できます。AI要約や分類はありませんが、記事タイトルとリンクは確認可能です。後からAPIキーを追加すれば、すべてのAI機能が有効になります。

## トラブルシューティング

| 問題 | 解決方法 |
|------|---------|
| `python`が見つからない | **「Add to PATH」** にチェックを入れてPythonを再インストール |
| `pip`が見つからない | `python -m pip install -r requirements.txt`を試す |
| `streamlit`が見つからない | `pip install streamlit`を実行 |
| 「APIキー未設定」の警告 | Step 6に従いAPIキー入りの`.env`ファイルを作成 |
| 記事が表示されない | まず「Collect」をクリック、1分待ってから「AI Process」 |
| AI機能が動作しない | `.env`ファイルが存在し、有効なAPIキーが記入されているか確認 |
| カテゴリが0件 | `python scripts/reclassify.py`を実行 |
| ポート6601が使用中 | `streamlit run app.py --server.port 7777`を使用 |
| 収集が遅い | 正常です — 74ソースは約60秒かかります |

**APIキーでよくある間違い:**
- キーの前後に余分なスペース → スペースをすべて削除
- 期限切れのキーを使用 → 新しいキーを発行
- `.env`ファイルの保存を忘れた → メモ帳で必ず保存をクリック
