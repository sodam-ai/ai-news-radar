## これは何？

AI News Radarは、世界中のAIニュースを**自動収集**し、AIが**要約・分類・重要度評価**を行う個人用ダッシュボードです。**35のLLMプラットフォーム**対応（ほとんど無料）、**Web＋デスクトップ**両対応、**5つのSNS**への自動投稿も可能です。

## 主な機能（45以上）

| カテゴリ | 機能 |
|---------|------|

## はじめ方

> **プログラミング経験不要です。**

### ステップ1：Pythonインストール
[python.org/downloads](https://www.python.org/downloads/) → **"Add Python to PATH"** を必ずチェック

### ステップ2：プロジェクトダウンロード
```
git clone https://github.com/sodam-ai/ai-news-radar.git
cd ai-news-radar
```

### ステップ3：パッケージインストール
```
pip install -r requirements.txt
```

### ステップ4：APIキー取得（無料）
[console.groq.com/keys](https://console.groq.com/keys) → サインアップ → キー作成 → コピー

### ステップ5：APIキー設定
`.env.example` を `.env` にコピー → メモ帳で開く → `GROQ_API_KEY=gsk_あなたのキー` を入力

### ステップ6：アプリ起動
```
streamlit run app.py
```
ブラウザが **http://localhost:6601** を自動的に開きます。

### ステップ7：初回使用
1. **"収集"** クリック → 74ソースからニュース収集
2. **"AI処理"** クリック → 全記事AI分析
3. **"ブリーフィング生成"** クリック → 今日のTOP 5生成

## トラブルシューティング

| 問題 | 解決 |
|------|------|
| `pip` が見つからない | Python再インストール、"Add to PATH"チェック |
| 記事が表示されない | 「収集」→「AI処理」の順にクリック |
| ポート使用中 | `--server.port 7429` オプション使用 |
