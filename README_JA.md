## AI News Radarとは？

AI News Radarは、世界中のAI関連ニュースを**自動的に収集**し、Google Gemini AIが**要約・分類・重要度を付ける**個人用ダッシュボードです。

**簡単に言うと：** 毎日15以上のニュースサイトを巡回する代わりに、このアプリが代わりにやって、重要なものだけを表示します。

---

## 主な機能

| 機能 | 説明 |
|------|------|
| 自動収集 | 15のプリセットソースからニュースを自動収集（TechCrunch、The Verge、MIT Tech Reviewなど） |
| AI要約 | 各記事を韓国語3行に要約 |
| スマート分類 | ツール/研究/トレンド/チュートリアル/ビジネスのカテゴリに自動分類 |
| 重要度スコア | 各記事に1〜5つ星の評価 |
| 感情分析 | 各記事をポジティブ/ニュートラル/ネガティブに分類 |
| 今日のブリーフィング | 毎日「今日のAIニュースTOP 5」を自動生成 |
| 重複統合 | 同じニュースを複数のメディアが報道した場合、1つにまとめる |
| キーワードウォッチリスト | 追跡キーワードを含むニュースをハイライト（例：「Claude」「GPT」） |
| アプリ内リーダー | 広告なしでダッシュボード内で原文を閲覧 |
| ダークモード | ダーク/ライトテーマの切り替え |
| エクスポート | ブリーフィングと記事をMarkdownまたはPDFでダウンロード |
| 画像分析 | ニュース内のチャート/インフォグラフィックをAIが分析 |
| リアルタイム更新 | ダッシュボードが5分ごとに自動更新 |
| タイムラインビュー | 時系列でニュースを閲覧（今日/昨日/今週） |
| Context Caching | システムプロンプトをキャッシュしてAPIトークンコストを最大90%削減 |
| スマートルーティング | 簡単な作業はFlash-Lite、高品質作業はFlashに自動振り分け |

## はじめ方（初心者向けステップバイステップガイド）

> **プログラミング経験は一切不要です。** 各ステップを順番に進めてください。

### ステップ1：Pythonをインストール

Pythonはこのアプリを動かすプログラミング言語です。

1. [python.org/downloads](https://www.python.org/downloads/) にアクセス
2. 大きな黄色い **「Download Python 3.xx」** ボタンをクリック
3. ダウンロードしたファイルを実行
4. **重要：** インストーラー下部の **「Add Python to PATH」** チェックボックスを必ずチェック
5. **「Install Now」** をクリック

**確認方法：** コマンドプロンプトを開き（`Win + R` → `cmd` と入力 → Enter）、以下を入力：
```
python --version
```
`Python 3.13.x` のような表示が出れば成功です！

### ステップ2：プロジェクトをダウンロード

**方法A：Gitを使う（推奨）**

Gitがインストールされている場合：
```
git clone https://github.com/sodam-ai/ai-news-radar.git
cd ai-news-radar
```

**方法B：直接ダウンロード**

1. [GitHubリポジトリページ](https://github.com/sodam-ai/ai-news-radar) にアクセス
2. 緑色の **「Code」** ボタンをクリック
3. **「Download ZIP」** をクリック
4. ZIPファイルを任意のフォルダに解凍

### ステップ3：必要なパッケージをインストール

コマンドプロンプトを開き、プロジェクトフォルダに移動して実行：
```
cd プロジェクトフォルダのパス\ai-news-radar
pip install -r requirements.txt
```

> **これは何をするの？** アプリの実行に必要なツール（ライブラリ）を自動的にダウンロードします。約1〜2分かかります。

### ステップ4：Gemini APIキーを取得（無料）

AI機能にはGoogle Gemini APIキーが必要です。**完全無料**です。

1. [aistudio.google.com/apikey](https://aistudio.google.com/apikey) にアクセス
2. Googleアカウントでログイン
3. **「Create API Key」**（APIキーを作成）をクリック
4. 生成されたキーをコピー（`AIzaSy...` の形式）

### ステップ5：APIキーを設定

1. プロジェクトフォルダで `.env.example` ファイルを探す
2. コピーして `.env` にリネーム
3. `.env` をテキストエディタ（メモ帳でOK）で開く
4. `your_gemini_api_key_here` を実際のAPIキーに置き換え：

```
GEMINI_API_KEY=AIzaSyここに実際のキーを入力
```

5. ファイルを保存

> **セキュリティ注意：** `.env` ファイルには秘密のAPIキーが含まれています。GitHubへのアップロードから自動的に除外されます。他の人と共有しないでください。

### ステップ6：アプリを起動

```
streamlit run app.py
```

ブラウザが自動的に **http://localhost:6601** を開きます。

完了です！AI News Radarが起動しました！

## 使い方

### 初回使用時

1. サイドバーの **「収集」** ボタンをクリック → ニュース収集
2. **「AI処理」** ボタンをクリック → 収集された記事をAI分析
3. **「ブリーフィング生成」** をクリック → 今日のTOP 5要約を生成

### 毎日の使い方

アプリは60分ごとに自動でニュースを収集します。ダッシュボードを開いて：

- **ブリーフィングタブ**でざっと確認
- **ニュースリスト**で詳しく読む
- サイドバーの**フィルター**で関心カテゴリ/感情のみ表示

### 機能ガイド

| やりたいこと | 方法 |
|-------------|------|
| ニュースソースを追加 | サイドバー > ソース管理 > 名前とRSS URLを入力 > 「ソース追加」クリック |

| 原文を読む | 記事タイトルをクリック（新しいタブ）または「詳しく見る」>「原文を取得」 |
| PDFでエクスポート | ブリーフィングタブまたはニュースタブ > 「エクスポート」 > PDFを選択 |
| Markdownでエクスポート | ブリーフィングタブまたはニュースタブ > 「エクスポート」 > Markdownを選択 |

| カテゴリフィルター | サイドバー > フィルターセクション > カテゴリを選択 |
| 感情フィルター | サイドバー > フィルターセクション > 感情を選択 |

## トラブルシューティング

| 問題 | 解決方法 |
|------|----------|
| `pip` コマンドが見つからない | Pythonを再インストールし「Add Python to PATH」をチェック |
| `streamlit` コマンドが見つからない | 実行：`pip install streamlit` |
| 「GEMINI_API_KEYが設定されていません」警告 | `.env` ファイルを作成したか確認（`.env.example` ではない） |
| 記事が表示されない | 「収集」を先にクリック、次に「AI処理」をクリック |
| ポート6601が使用中 | 他のStreamlitを終了するか `.streamlit/config.toml` でポート変更 |
| PDFエクスポートが失敗 | Windows環境でのみ動作（韓国語サポートにWindowsフォント使用） |
| 記事は収集されたがAI分析されない | `.env` ファイルのGEMINI_API_KEYが正しいか確認 |
| 起動時にエラー | `pip install -r requirements.txt` を再実行して不足パッケージを確認 |

## 無料APIの制限

AI News RadarはGeminiの無料枠で**完全無料**で運用可能です：

| モデル | 無料制限 | 用途 |
|--------|---------|------|
| Gemini Flash-Lite | 1,000回/日 | 分類、タグ、感情分析 |
| Gemini Flash | 250回/日 | 要約、ブリーフィング、画像分析 |

スマートルーティングシステムが、簡単な作業は軽量モデルで、複雑な作業は高性能モデルで自動的に振り分けます。
