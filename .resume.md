## 발생한 문제 / 원인 / 해결 여부

| 문제 | 원인 | 해결 |
|------|------|------|
| chromadb DefaultEmbeddingFunction이 Google API 호출 | chromadb 1.5.x에서 DefaultEmbeddingFunction이 Google Embedding API를 기본값으로 변경됨 | `ONNXMiniLM_L6_V2` 로컬 모델로 교체, `onnxruntime` 설치 ✅ |
| 이전 세션 Gemini API 키 만료 | API 키가 세션 간에 파기됨 | chromadb를 API 불필요 ONNX 방식으로 변경하여 회피 ✅ |
