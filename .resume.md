### Phase 2-A 작업 목표: SQLite 전환
```
목표: data/articles.json → data/radar.db (SQLite)
신규 파일: db/database.py (DB 초기화 + CRUD)
수정 파일: app.py, ai/batch_processor.py, ai/chat.py, bot/telegram_bot.py
마이그레이션: utils/migrate_json_to_sqlite.py (1회용 스크립트)

테이블 설계:
  articles (id TEXT PK, title, url, category, importance, sentiment,
            summary_text, tags, published_at, collected_at,
            ai_processed, is_primary, source)
  briefings (id TEXT PK, date, summary, top_articles_json, created_at)
  watchlist (id INTEGER PK, keyword, is_active, created_at)
```
